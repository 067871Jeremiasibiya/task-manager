# Task Manager - Kanban Board

A beautiful, interactive Kanban-style task management application built with React, TypeScript, and Tailwind CSS.

![Task Manager](https://images.unsplash.com/photo-1611224923853-80b023f02d71?w=800&h=400&fit=crop)

## Features

- Drag-and-drop task management (Kanban board)
- Three columns: To Do, In Progress, Done
- Add, edit, and delete tasks
- Priority levels (Low, Medium, High)
- Task statistics dashboard
- Smooth animations with Framer Motion
- Responsive design
- Local state management

## Tech Stack

- **React 18** - UI Framework
- **TypeScript** - Type Safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **UUID** - Unique ID generation
- **Vite** - Build tool

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open http://localhost:3002 in your browser
